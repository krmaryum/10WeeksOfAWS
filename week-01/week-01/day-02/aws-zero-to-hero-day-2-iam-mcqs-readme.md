# README – AWS Zero To Hero Day 2 IAM MCQs

## Overview

This HTML quiz contains 25 multiple-choice questions based on the **AWS Zero To Hero – Day 2 IAM Fundamentals** PDF.

## Topics Covered

- IAM users, groups, and roles
- Authentication vs authorization
- IAM policies
- AWS managed, customer managed, and inline policies
- JSON policy structure
- Effect, Action, Resource, Statement, Condition
- Least privilege
- Permission boundaries
- IAM labs for S3, EC2, and Billing access
- Day 2 safety notes

## Features

- 25 MCQs
- 25-minute timer
- Score checking
- Correct/wrong answer highlighting
- Short explanation for each question
- Topic-wise result breakdown
- Re-attempt button
- Answer choices shuffle on each re-attempt

## Suggested Goal

Try to score 80% or above.

## Recommended Revision Focus

- IAM user vs group vs role
- Authentication vs authorization
- Managed policy vs inline policy
- JSON policy elements
- Explicit Deny wins
- Least privilege
- Permission boundary does not grant access
- Use IAM roles for AWS services like EC2
