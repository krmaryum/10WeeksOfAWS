This HTML quiz is created for **AWS Day 1 and Day 2 revision**.

It covers:

- AWS Cloud foundations
- Cloud computing basics
- AWS global infrastructure
- Shared Responsibility Model
- AWS account setup
- Billing basics
- Root user vs IAM user
- MFA
- IAM users, groups, roles, and policies
- Authentication vs authorization
- Least privilege
- Permission boundaries
- IAM best practices

## How to Open the Quiz

### Option 1: Open directly in browser

   - Google Chrome

The quiz will open in your browser.

## How to Use

1. Open the quiz HTML file.
2. Read each question carefully.
3. Select one answer for each question.
4. Watch the timer.
5. Click **Submit Quiz**.
6. Review:
   - Your score
   - Correct answers
   - Wrong answers
   - Explanations
7. Click **Re-attempt Quiz** to practice again.
