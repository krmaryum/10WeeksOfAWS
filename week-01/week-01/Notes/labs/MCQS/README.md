# README – AWS Week 1 MCQs Quiz

## Overview

This folder contains a 25-question interactive HTML quiz for **AWS Week 1** revision.

The quiz covers the main topics learned during Week 1, including:

```text
AWS account security
IAM users, groups, roles, and policies
S3 buckets and objects
S3 bucket ARN and object ARN
EC2 read-only access
Billing read-only access
STS and temporary credentials
Switch Role
GitHub OIDC with AWS
GitHub Actions workflow authentication
```

---

## Quiz File

```text
aws-week-1-25-mcqs-quiz.html
```

---

## Features

The quiz includes:

```text
25 MCQs
Answer checking
Final score
25-minute timer
Short explanation for each answer
Show answers button
Re-attempt option
Questions shuffle on re-attempt
Answer options shuffle on re-attempt
```

---

## How to Use

Open the HTML file in any browser:

```text
aws-week-1-25-mcqs-quiz.html
```

Then:

1. Read each question carefully.
2. Select one answer for each question.
3. Click **Submit Quiz**.
4. Check your score and explanations.
5. Click **Re-attempt / Shuffle** to practice again.
